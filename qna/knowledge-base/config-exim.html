How do I configure exim to work with GoodCrypto?

Exim4 support is experimental.

  1. Edit /etc/exim4/update-exim4.conf.conf so the dc_local_interfaces line includes ; GC_PRIVATE_SERVER_IP.10028 before the closing quote mark. Change GC_PRIVATE_SERVER_IP to the IP address of your GoodCrypto private server.

     

  2. Download the GoodCrypto router and save it in /etc/exim4/conf.d/router directory

     

  3. Download the GoodCrypto transport and save it in /etc/exim4/conf.d/transport directory

     

  4. Restart exim and you're done

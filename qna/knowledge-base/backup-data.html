How do I backup the data from my server?

Your backup procedure depends on where you've installed your GoodCrypto private server.

Stand alone computer
It's best to back up the entire drive where you installed GoodCrypto. This will make restoration quick and easy. If you are limited in space and you have physical access, then you can just backup /media/drive/persistence.
Virtual machine
You simply need to back up the the VDI file that you created when you set up the virtual machine.

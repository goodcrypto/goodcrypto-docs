How do I simplify managing keys?

Use the defaults for auto exchanging keys and creating keys.

  1. Connect to the Goodcrypto Server website.
  2. Sign in
  3. Click on Mail in the menu
  4. Click on Options. This button only appears if you sign in as an administrator.
  5. Click on the MTA address
  6. Add a check mark to Auto-exchange keys if there isn't one
  7. Add a check mark to Create private keys if there isn't one
  8. Click Save

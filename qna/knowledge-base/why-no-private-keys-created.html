Why aren't private keys created?

There are a few possibilities so check:

  1. Is GoodCrypto Mail active? Go to the GoodCrypto Server's website and click the Status menu item.
  2. Do you have the Mail Options set to create private keys? If not, enable this option from your GoodCrypto Server's website. Click the Mail menu item, and then click "Options".
  3. If there was an error while the key was being created, then a semaphore might have been left in the queue.

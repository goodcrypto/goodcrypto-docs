How do I run my GoodCrypto ISO on a new computer?

To reduce the possibility of hidden malware, GoodCrypto will reformat the system data drive.

  1. Save the iso to a bootable device. We recommend you burn to readonly media such as a DVD. If you choose to use a usb drive, you can just copy the iso straight to the drive.
  2. System requirements:
    • Intel 64 bit processor with virtual machine support
    • 4+ GB RAM + mail needs (8 GB or more suggested)
    • 10 GB drive + mail space (500 GB or more suggested)
      The drive must be empty.
    • Bootable from ISO (DVD recommended, USB, etc.)
    • DHCP access
    • Internet access
  3. Back up any data from the hard drive. GoodCrypto will reformat the drive to reduce the possibility of hidden malware.
  4. Boot the new computer from the iso. GoodCrypto needs a dedicated data drive. If it can't find an empty one to format, it will say so and not boot. You might see an error message about the wrong file type which you can ignore. GoodCrypto will automatically format the drive with the correct type. On the first boot, you may notice a delay. GoodCrypto is reformatting the empty drive and creating unique keys.
  5. Configure and maintain your GoodCrypto server on your private website. Browse to the new computer's IP address using port 8443 if you're using https, or 8080 if you're using http. Examples: http://199.18.0.50:8080 or https://199.18.0.50:8443. We recommend you use https, but remember that your private server has a self-signed cert so you'll need to have your browser accept the cert. If your browser says it can't contact the site, wait a few minutes and try again.
  6. Enter the administrator's address for the domain you want to protect on your GoodCrypto private server. This address will be used to find your mail server, and to receive admin messages from your GoodCrypto private server. You may also opt to receive security alerts from GoodCrypto at this address.
  7. Follow the rest of the instructions provided on your GoodCrypto private server.

We strongly recommend that you:

  1. Maintain your GoodCrypto Server in a secure location.
  2. Limit access to authorized users.
  3. Maintain a correctly configured firewall.

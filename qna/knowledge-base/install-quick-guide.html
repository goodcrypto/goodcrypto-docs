How do I install my GoodCrypto private server?

  1. Download the ISO
  2. Install the ISO on a new computer or in a virtual machine
  3. Use your browser to go to the IP address at port 8080 for the computer where you installed the ISO. If your browser reports the site it not accessible, wait a few minutes and try reloading.
      It's better to go connect via https with port 8443, but remember that your private server has a self-signed cert so you'll need to have your browser accept the cert.
  4. Enter an email address that uses the domain you want to protect on your GoodCrypto private server.
    This address will be used to receive notices (e.g., error messages) from your GoodCrypto private server. You may also opt to receive security alerts from GoodCrypto at this address.
  5. Follow the instructions on your private server to finish the integration.

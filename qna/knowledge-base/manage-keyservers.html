How do I manage the keyservers?

  1. Connect to the Goodcrypto private server website.
  2. Sign in with the administrator credentials
  3. Click on Mail in the menu
  4. Click on Keyservers button. This button only appears if you're logged in as the administrator.
  5. You can add, change, or delete keyservers. If you don't want to delete a keyserver, but no longer use it, then simply remove the Active check mark while editing it.

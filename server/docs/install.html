Installation

After installing the preconfigured ISO, you simply need to customize a few fields and integrate it with your mail server and browsers.

 

 

Quick Guide

  1. Download the ISO
  2. Install the ISO on a new computer or in a virtual machine
  3. Use your browser to go to the IP address at port 8080 for the computer where you installed the ISO. If your browser reports the site it not accessible, wait a few minutes and try reloading.
      It's better to go connect via https with port 8443, but remember that your private server has a self-signed cert so you'll need to have your browser accept the cert.
  4. Enter an email address that uses the domain you want to protect on your GoodCrypto private server.
    This address will be used to receive notices (e.g., error messages) from your GoodCrypto private server. You may also opt to receive security alerts from GoodCrypto at this address.
  5. Follow the instructions on your private server to finish the integration.

Configure Your Browsers

We strongly recommend that you ask your administrator to help configure your browser using the proxy settings:

  • Chrome
  • Firefox

 

Proxy Settings

HTTP Proxy: Your GoodCrypto private server IP address 8398
SSL Proxy: Your GoodCrypto private server IP address 8398
FTP Proxy: leave blank 0
SOCKS Host: Your GoodCrypto private server IP address 9350

 

Chrome

  1. Click on the menu and select Settings
  2. Click on the Show advanced settings link
  3. Under the Network section, click Change proxy settings
  4. Select "Manual proxy configuration"
  5. Fill in the proxy settings
  6. Uncheck "Use this proxy for all protocols" after "HTTP Proxy".
  7. Select "SOCKS v5".

 

 

Firefox

Edit / Preferences / Advanced / Network / Settings

  1. Select "Manual proxy configuration"
  2. Fill in the proxy settings
  3. Uncheck "Use this proxy for all protocols" after "HTTP Proxy".
  4. Select "SOCKS v5".
